제이쿼리 선택자(selector)

제이쿼리에서는 css에서 사용하던 선택자를 활용하여 서식을 적용할 수 있는데, 제이쿼리는 css보다 좀 더 세밀하게 선택할 수 있고 그 개수도 많다.

http://oscarotero.com/jquery/ - API 참고 사이트의 왼쪽 'SELECTORES'에서 확인

제이쿼리 선택자 사용하여 서식 적용하기

  • 아이디 선택자를 사용하여 글자색 변경
  • 클래스 선택자를 사용하여 배경색 변경
  • 태그 선택자를 사용하여 글자크기 변경
  • 그룹선택자를 사용하여 테두리 적용
  • 자손, 후손 선택자를 사용하여 배경색, 글자모양, 글자색상 적용
  • 구조 선택자를 사용하여 홀수, 짝수번째 배경색 적용
  • 등위 선택자를 사용하여 글자 밑줄긋기
  • 탐색 선택자를 사용하여 홀수, 짝수번째 글자모양 변경
  • 속성 선택자를 사용하여 글자색, 굵기 변경

제이쿼리에서 css서식 적용하는 방법

    .css()메소드 사용법
    서식이 1개일 때       .css('속성', '값');
    서식이 2개 이상일 때  .css('속성', '값').css('속성', '값').css('속성', '값');   <=비효율적이라 잘 안씀
    서식이 2개 이상일 때  .css({'속성': '값', '속성': '값', '속성': '값'});
