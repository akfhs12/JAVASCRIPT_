제이쿼리 효과(effect)

사용자가 지정한 요소를 선택하여 숨기기, 나타내기, 페이드 효과, 슬라이드 다운 업, 애니메이션 기능을 사용하기 위한 메서드들

효과 메서드 종류

  • show(); - 선택한 요소를 보이게 함
  • hide(); - 선택한 요소를 숨김
  • toggle(); - 선택한 요소를 보이게하거나 숨김
  • slideDown(); - 선택한 요소를 아래로 펼침
  • slideUp(); - 선택한 요소를 위로 접음
  • slideToggle(); - 선택한 요소를 접거나 아래로 펼친다
  • fadeIn(); - 선택한 요소를 서서히 보이게 한다
  • fadeOut(); - 선택한 요소를 서서히 숨긴다
  • fadeToggle(); - 선택한 요소를 서서히 보이게 하거나 숨긴다
  • fadeTo(시간, 투명도); - 선택한 요소를 지정한 시간동안 투명하게 설정
