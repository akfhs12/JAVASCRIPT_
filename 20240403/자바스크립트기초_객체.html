자바스크립트 기초_객체(object)

객체(object)란?
1. 이름(key)과 값(value)으로 구성된 속성을 담아놓은 컨테이너
2. 객체에 변수를 선언하면 속성(property)을 사용할 수 있다. 예)document.getElementById('아이디명');
3. 객체에 함수를 선언하면 메서드(method)를 사용 할 수 있다. 예)document.write('내용출력')

1. 객체의 종류

객체는 내장 객체와 사용자 정의 객체로 분류. 내장 객체는 자바스크립트에 미리 정해놓은 객체이다.

  • 문서객체(document)
  • 윈도우, 브라우저객체(window)
  • 수학(Math)객체
  • 날짜(Date)객체
  • 배열(Array)객체
  • 문자열(String)객체

2 사용자 정의 객체

내장 객체를 제외한, 사용자가 원하는 내용을 토대로 직접 객체를 선언하여 작성하는 방법

휴대폰 정보 입력하기

제조회사 :

모델명 :

색상명 :

출력 예) 나의 핸드폰 제조 회사는 '000'이고, 모델명은 '000'이며, 색상은 '000'입니다.

개인정보 입력

성 :

이름 :

나이 :

키 :

몸무게 :
