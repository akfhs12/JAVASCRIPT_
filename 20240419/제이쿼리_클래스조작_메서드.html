제이쿼리 문서객체조작_클래스조작

사용자가 선택한 요소에 클래스를 추가하거나 삭제, 또는 클래스가 없는 요소를 찾아 서식을 적용하는 메소드

  • addClass() - 선택한 요소에 클래스 서식을 추가
  • removeClass() - 선택한 요소에 클래스 서식을 제거
  • toggleClass() - 선택한 요소에 클래스 서식을 추가/제거
  • hasClass() - 선택한 요소에 해당 클래스 이름이 있는지 여부를 판단하여 true, false값으로 표시
  • class() - 사용자가 선택한 요소에 서식을 적용
