제이쿼리 문서 객체(DOM) 조작

문서객체조작은 DOM(문서객체모델), HTML 문서 안에서 제이쿼리를 통해 모든 요소와 내용의 속성값 변경, 추가, 삭제 등을 할 수 있는 것을 말한다

https://oscarotero.com/jquery 웹사이트 API에서 'ATTRIBUTES', 'MANIPULATION' 카테고리에서 메소드 확인 가능

문서 내용을 변경하기 위한 메서드 종류

  • $('대상').text() - 텍스트 내용 변경을 위한 메서드
  • $('대상').html() - 태그와 텍스트 내용 변경을 위한 메서드
  • $('대상').attr() - 선택한 요소의 속성이나 속성값을 변경하기 위한 메서드
  • $('대상').val() - 선택한 요소의 폼태그 value 값 변경
  • $('대상').empty() - 선택한 요소의 내용 지우기
  • $('대상').addClass() - 클래스 추가. js에서는 classList.add
  • $('대상').removeClass() - 클래스 제거. js에서는 classList.remove
  • $('대상').hasClass() - 클래스 존재 여부 확인
  • $('대상').toggleClass() - 클래스 존재 여부 확인
  • $('내용').appendTo('대상') - 부모내용 마지막 위치로 이동(자리바꾸기)
  • $('내용').prependTo('대상') - 부모내용 처음 위치로 이동(자리바꾸기)
  • $('A').insertAfter('B') - B요소의 뒤에 A를 배치
  • $('A').insertBefore('B') - B요소의 앞에 A를 배치
