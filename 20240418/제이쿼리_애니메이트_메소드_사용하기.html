제이쿼리_애니메이트_메소드_사용하기

제이쿼리에서는 효과와 애니메이트 메소드를 사용하여 사용자가 선택한 요소를 다양하게 움직이도록 만들 수 있다

1. animate 메서드 사용법

$('선택자').animate({'속성':'값'}, 시간, 가속도, 콜백함수);

  • 속성: css속성과값을 말하며, '값'에는 자바스크립트 변수나 연산자 활용 가능
  • 시간: 1/1000 => 1초값을 의미
  • 콜백함수: callback function 으로서 애니메이션 동작이 끝나고 나서 다음 동작을 이어서 실행하고자 할 때 사용
