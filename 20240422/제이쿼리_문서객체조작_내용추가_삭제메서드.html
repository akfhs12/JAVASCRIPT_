제이쿼리 문서객체조작_내용_추가_삭제메서드

DOM(문서객체모델)에서 문서안의 요소에 내용을 추가하거나 삭제할 수 있다.

클래스조작(addClass, removeClass, toggleClass, hasClass)

속성조작(attr)

내용조작(text, html)

  • append(); 사용자가 선택한 요소를 기준으로 안쪽 맨 뒤쪽에 내용 추가
  • prepend(); - 사용자가 선택한 요소를 기준으로 안쪽 맨 앞쪽에 내용 추가
  • after(); - 사용자가 선택한 요소를 기준으로 바깥쪽 맨 뒤에 내용 추가
  • before(); - 사용자가 선택한 요소를 기준으로 바깥쪽 맨 앞에 내용 추가

insertAfter(), insertBefore()는 , after(), before()메서드와 기능은 같지만 사용법이 다르다

예)
$('A').after('B'); - A요소의 바깥쪽으로 뒤에 B요소를 추가한다 $('B').insertafter('A'); - A요소의 바깥쪽으로 뒤에 B요소를 추가한다

문서객체조작 - 내용을 삭제하기 위한 메서드(remove)

사용자가 선택한 요소의 내용만 삭제하거나 아예 태그 요소를 삭제할 수도 있다

  • remove() - 사용자가 선택한 요소를 제거
  • empty() - 사용자가 선택한 요소의 내용만 제거
아래 remove 버튼을 클릭하면 이 박스요소가 제거
아래 empty 버튼을 클릭하면 이 박스 요소 내의 내용만 제거
조건식 만들기 - 태그 존재 여부 판단
javascript - if(document.getElementById('태그명')){존재하면 실행}
jQuery - if($('.class').length > 0){존재하면 실행}
jQuery - if($('.class').hasClass('.class')){존재하면 실행}
