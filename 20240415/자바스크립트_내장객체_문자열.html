자바스크립트 내장객체_문자열 객체

자바스크립트의 내장 객체 중에서 활용도가 많은 객체이며, 문자열 값을 인덱스 번호를 통해 값을 찾거나 다른 문자로 변경도 가능

아이디, 패스워드 입력창 문자열 개수 체크, 단문 메세지 문자열 개수 체크, 패스워드 문자열 찾아서 다른 문자열로 변경, 이미지 갤러리 등

문자열 객체는 다른 내장 객체와는 다르게 'new'키워드 생략 가능

문자열 객체의 메서드 종류

  • replace - 문자열 치환
  • split - 문자열 분리
  • substring - 인덱스 번호 구간부터 문자데이터 찾아줌
  • length - 문자열 개수를 체크해줌
  • indexof - 문자열과 일치하는 인덱스 번호를 반환

실습1. url 주소를 사용하여 원하는 문자열로 출력

실습2. textarea 태그를 사용하여 사용자가 입력한 글자수를 체크하여 더이상 글자가 써지지 않도록 하며 몇글자를 입력하였는지 체크하기

글자수 체크하기 - 댓글, 답변글, 상품평 남기기, 실시간 문의글 등에 활용
