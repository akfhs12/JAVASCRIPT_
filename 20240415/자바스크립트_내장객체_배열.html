내장객체(object) - 배열Array

배열(Array) 객체란?
연관된 데이터를 하나처럼 관리하고 싶을 때 사용
복수의 데이터를 제어할 수 있는 다양한 함수를 제공
배열의 함수를 이용하여 데이터의 편리한 조작 가능

연관된 정보 - 예) 회원 정보(이름, 전화번호, 아이디, 패스워드, 성별), 쇼핑정보, 예약정보 등

1. 배열 객체 선언

방법1) let array = new Array(data1, data2, data3...dataN);

예) let array = new Array('사과', '바나나', '수박', '딸기');

방법2) let array = [data1, data2, data3...dataN];

예) let array = ['사과', '바나나', '수박', '딸기'];

2. 배열 객체 호출 array[n]

인덱스 번호를 사용하여 호출하며 배열 인덱스 번호는 0부터 시작

3. 배열 객체 메서드

concat = 기존 배열에 다른 배열 추가

filter - 배열 요소 중에서 주어진 필터링 함수에 대해 true인 요ㅗ만 골라 새로운 배열을 만듦

forEach - 배열의 모든 요소에 대해 주어진 함수를 실행

indexOf - 주어진 값과 일치하는 값이 있는 배열 요소의 첫 인덱스를 찾아줌
